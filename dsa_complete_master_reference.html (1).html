  • String comparison: .equals() not ==
  • StringBuilder for loops — O(1) append vs O(n) for String +

👁️ Visual

☕ Code Templates

Java
📌 Kadane's Algorithm
public int maxSubArray(int[] nums) {
    int maxSum = nums[0], currentSum = nums[0];
    for (int i = 1; i < nums.length; i++) {
        currentSum = Math.max(nums[i], currentSum + nums[i]);
        maxSum = Math.max(maxSum, currentSum);
    }
    return maxSum;
}
// Time: O(n)  Space: O(1)
📌 Valid Anagram
public boolean isAnagram(String s, String t) {
    if (s.length() != t.length()) return false;
    int[] count = new int[26];
    for (int i = 0; i < s.length(); i++) {
        count[s.charAt(i) - 'a']++;
        count[t.charAt(i) - 'a']--;
    }
    for (int c : count) if (c != 0) return false;
    return true;
}
// Time: O(n)  Space: O(1)

📌 Two Sum — the classic pattern
public int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int complement = target - nums[i];
        if (map.containsKey(complement)) return new int[]{map.get(complement), i};
        map.put(nums[i], i);
    }
    return new int[]{};
}
// Time: O(n)  Space: O(n)
🧠 Memory Hook

"FIND = HashMap." Store what you've seen, check if complement exists.

📌 Two Sum II — Sorted Array
public int[] twoSumSorted(int[] numbers, int target) {
    int left = 0, right = numbers.length - 1;
    while (left < right) {
        int sum = numbers[left] + numbers[right];
        if (sum == target) return new int[]{left+1, right+1};
        else if (sum < target) left++;
        else right--;
    }
    return new int[]{};
}
// Time: O(n)  Space: O(1)

🎯 Pattern Recognition

Identify First
🔑 Keywords
"subarray of size k""longest substring""minimum window""at most k distinct"
💡 InsightBrute force checks every subarray O(n²). Sliding window reuses prior computation — O(n).

📖 Theory + Analogy

Theory
🌍 Analogy

Fixed window = train window of fixed width sliding past scenery. Variable window = elastic band — expand right, shrink left when condition breaks.

👁️ Visual

Visual
"a b c a b c b b" — longest substring no repeat [a]→1 [a b]→2 [a b c]→3 [b c a]→'a' repeat, shrink→3 Answer = 3

☕ Code Templates

Java
📌 Longest Substring Without Repeating
public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> map = new HashMap<>();
    int maxLen = 0, left = 0;
    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        if (map.containsKey(c) && map.get(c) >= left) left = map.get(c) + 1;
        map.put(c, right);
        maxLen = Math.max(maxLen, right - left + 1);
    }
    return maxLen;
}
// Time: O(n)  Space: O(min(n,alphabet))
🧠 Memory Hook

EXPAND → VALIDATE → SHRINK. Right always expands; left shrinks only when invalid.

📖 Theory + Analogy

Theory
🌍 Analogy

👁️ Visual

Visual

☕ Code Templates

Java
📌 Reverse Linked List
public ListNode reverseList(ListNode head) {
    ListNode prev = null, curr = head;
    while (curr != null) {
        ListNode next = curr.next;
        curr.next = prev; prev = curr; curr = next;
    }
    return prev;
}
// Time: O(n)  Space: O(1)
📌 Merge Two Sorted Lists — Dummy Head
public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    ListNode dummy = new ListNode(0), tail = dummy;
    while (l1 != null && l2 != null) {
        if (l1.val <= l2.val) { tail.next = l1; l1 = l1.next; }
        else { tail.next = l2; l2 = l2.next; }
        tail = tail.next;
    }
    tail.next = (l1 != null) ? l1 : l2;
    return dummy.next;
}
// Time: O(n+m)  Space: O(1)

📌 Valid Parentheses
public boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c=='('||c=='['||c=='{') stack.push(c);
        else {
            if (stack.isEmpty()) return false;
            char top = stack.pop();
            if (c==')'&&top!='(') return false;
            if (c==']'&&top!='[') return false;
            if (c=='}'&&top!='{') return false;
        }
    }
    return stack.isEmpty();
}
// Time: O(n)  Space: O(n)

☕ Code Templates

📌 BFS Level Order Traversal
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;
    Queue<TreeNode> queue = new ArrayDeque<>();
    queue.offer(root);
    while (!queue.isEmpty()) {
        int size = queue.size();
        List<Integer> level = new ArrayList<>();
        for (int i=0; i<size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
// Time: O(n)  Space: O(n)

Monotonic Stack

🎯 Pattern Recognition

Identify First
🔑 Keywords
"next greater element""daily temperatures""largest rectangle histogram"

📖 Theory + Analogy

Theory
🌍 Analogy

People waiting facing forward — when a taller person arrives, shorter ones in front who are "blocked" leave (pop). Stack stays monotonic.

👁️ Visual

Visual
[2,1,2,4,3] → Next Greater = [4,2,4,-1,-1]

☕ Code Templates

Java
📌 Next Greater Element
public int[] nextGreaterElement(int[] nums) {
    int[] result = new int[nums.length];
    Arrays.fill(result, -1);
    Deque<Integer> stack = new ArrayDeque<>();
    for (int i=0; i<nums.length; i++) {
        while (!stack.isEmpty() && nums[i] > nums[stack.peek()]) {
            result[stack.pop()] = nums[i];
        }
        stack.push(i);
    }
    return result;
}
// Time: O(n)  Space: O(n)
🧠 Memory Hook

"Pop while smaller, push always." Store indices, not values.

📌 Lowest Common Ancestor
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    if (root == null || root == p || root == q) return root;
    TreeNode left = lowestCommonAncestor(root.left, p, q);
    TreeNode right = lowestCommonAncestor(root.right, p, q);
    if (left != null && right != null) return root;
    return left != null ? left : right;
}
// Time: O(n)  Space: O(h)

📌 Validate BST — range-passing pattern
private boolean validate(TreeNode node, Integer min, Integer max) {
    if (node == null) return true;
    if (min != null && node.val <= min) return false;
    if (max != null && node.val >= max) return false;
    return validate(node.left, min, node.val) && validate(node.right, node.val, max);
}
// Time: O(n)  Space: O(h)

☕ Code Templates

Java
📌 Kth Largest Element
public int findKthLargest(int[] nums, int k) {
    PriorityQueue<Integer> minHeap = new PriorityQueue<>();
    for (int num : nums) {
        minHeap.offer(num);
        if (minHeap.size() > k) minHeap.poll();
    }
    return minHeap.peek();
}
// Time: O(n log k)  Space: O(k)
🧠 Memory Hook

Kth largest → Min-heap of size k. Kth smallest → Max-heap of size k.

📌 Number of Islands — DFS Flood Fill
public int numIslands(char[][] grid) {
    int count = 0;
    for (int r=0; r<grid.length; r++)
        for (int c=0; c<grid[0].length; c++)
            if (grid[r][c] == '1') { count++; dfs(grid, r, c); }
    return count;
}
private void dfs(char[][] grid, int r, int c) {
    if (r<0||c<0||r>=grid.length||c>=grid[0].length||grid[r][c]!='1') return;
    grid[r][c] = '0';
    dfs(grid,r+1,c); dfs(grid,r-1,c); dfs(grid,r,c+1); dfs(grid,r,c-1);
}
// Time: O(rows×cols)  Space: O(rows×cols)

    int rootX = find(x), rootY = find(y);

👁️ Visual

☕ Code Templates

Java
📌 House Robber + Coin Change
public int rob(int[] nums) {
    int prev2=0, prev1=0;
    for (int num : nums) {
        int curr = Math.max(prev1, prev2+num);
        prev2 = prev1; prev1 = curr;
    }
    return prev1;
}

public int coinChange(int[] coins, int amount) {
    int[] dp = new int[amount+1];
    Arrays.fill(dp, amount+1); dp[0]=0;
    for (int i=1; i<=amount; i++)
        for (int coin : coins)
            if (coin<=i) dp[i] = Math.min(dp[i], dp[i-coin]+1);
    return dp[amount] > amount ? -1 : dp[amount];
}
// House Robber: O(n)/O(1)   Coin Change: O(amount×coins.length)/O(amount)

  • Match → diagonal+1: dp[i][j]=dp[i-1][j-1]+1

👁️ Visual

☕ Code Templates

Java
📌 Longest Common Subsequence
public int longestCommonSubsequence(String t1, String t2) {
    int m=t1.length(), n=t2.length();
    int[][] dp = new int[m+1][n+1];
    for (int i=1; i<=m; i++)
        for (int j=1; j<=n; j++)
            dp[i][j] = (t1.charAt(i-1)==t2.charAt(j-1))
                ? dp[i-1][j-1]+1 : Math.max(dp[i-1][j], dp[i][j-1]);
    return dp[m][n];
}
// Time: O(m×n)  Space: O(m×n)

Binary Search

🎯 Pattern Recognition

Identify First
🔑 Keywords
"sorted array""first/last occurrence""rotated sorted array""minimize the maximum"
🧠 Memory Hook

"Can I cut the search space in half via a yes/no check?" → binary search applies, even on the ANSWER not just an array.

📖 Theory + Analogy

Theory
🌍 Analogy

Guessing 1-100 with higher/lower feedback — each guess halves the space.

⚠️ GotchaNever (lo+hi)/2 — overflow risk. Use lo + (hi-lo)/2.

👁️ Visual

Visual
[1,3,5,7,9,11,13], target=9 mid=3→7<9→right; mid=5→11>9→left; mid=4→9 FOUND

☕ Code Templates

Java
📌 First/Last Occurrence — Boundary Search
public int findFirst(int[] nums, int target) {
    int lo=0, hi=nums.length-1, result=-1;
    while (lo <= hi) {
        int mid = lo + (hi-lo)/2;
        if (nums[mid] == target) { result = mid; hi = mid-1; }
        else if (nums[mid] < target) lo = mid+1;
        else hi = mid-1;
    }
    return result;
}
// Time: O(log n)  Space: O(1)

Backtracking

🎯 Pattern Recognition

Identify First
🔑 Keywords
"all subsets""all permutations""combination sum""N-Queens"

📖 Theory + Analogy

Theory
🌍 Analogy

Maze with erasable chalk marks — choose, explore, unchoose (erase) on dead ends.

👁️ Visual

Visual
Subsets of [1,2,3]: [] → [1],[2],[3] → [1,2],[1,3],[2,3] → [1,2,3] Total = 2^n

☕ Code Templates

Java
📌 Subsets — Power Set
private void backtrack(int[] nums, int start, List<Integer> path, List<List<Integer>> result) {
    result.add(new ArrayList<>(path));
    for (int i=start; i<nums.length; i++) {
        path.add(nums[i]);
        backtrack(nums, i+1, path, result);
        path.remove(path.size()-1);
    }
}
// Time: O(n×2^n)  Space: O(n)
🧠 Memory Hook

Subsets loop from `start` (order doesn't matter). Permutations loop from 0 + used[] (order matters).

📌 Non-overlapping Intervals
public int eraseOverlapIntervals(int[][] intervals) {
    Arrays.sort(intervals, (a,b) -> a[1]-b[1]);
    int count=0, lastEnd=Integer.MIN_VALUE;
    for (int[] interval : intervals) {
        if (interval[0] >= lastEnd) lastEnd = interval[1];
        else count++;
    }
    return count;
}

public void mergeSort(int[] arr, int lo, int hi) {
    if (lo >= hi) return;
    int mid = lo + (hi-lo)/2;

}

public int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

public List<Integer> sieveOfEratosthenes(int n) {
    boolean[] isComposite = new boolean[n+1];
    List<Integer> primes = new ArrayList<>();
    for (int i=2; i<=n; i++)
        if (!isComposite[i]) {
            primes.add(i);
            for (long j=(long)i*i; j<=n; j+=i) isComposite[(int)j] = true;
        }
    return primes;
}
// GCD: O(log min(a,b))   Sieve: O(n log log n)

Complexity Analysis

📖 The Big O Hierarchy — memorize this order

Theory
O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2^n) < O(n!) fastest ────────────────────────────────────────────► slowest
  • Time complexity — worst-case growth of runtime as input size n increases. Always state it for every solution.
  • Space complexity — auxiliary space (extra memory you allocate) vs total space (includes input). Interviewers usually mean auxiliary.
  • Amortized analysis — ArrayList's add() is O(1) amortized even though resizing is O(n), because resizes happen rarely (doubling strategy spreads the cost).
  • In interviews: always state complexity before and after any optimization — it signals you're thinking deliberately, not guessing.
🧠 Memory Hook

"Nested loop over n → O(n²). Halving each step → O(log n). Trying every subset → O(2^n). Trying every order → O(n!)." Recognize the shape of your code, not just count loops.
